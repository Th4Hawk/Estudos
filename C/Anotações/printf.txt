O printf é uma função disponivel na biblioteca de entrada e saida de dados do C (stdio.h), essa função permite imprimir textos no console, além de permitir argumentos adicionais como:

printf("O numero é: %d", 2);

O %d permite que variaveis decimais sejam mostradas no console, para mostrar uma cadeia de caracteres podemos usar o %s. Vale lembrar que para representar um caractere usamdo as aspas simples ('') e para representar uma cadeia de caracteres usamos as aspas duplas ("").

Tabela de codigos especiais:

\n -> Nova linha
\r -> Retorno do cursor
\t -> Tab
\b -> Retrocesso
\" -> Aspas 
\\ -> Barra
\f -> Salta pagina de formulario
\0 -> Nulo

Tabela de codigos para impressão formatada:

%e -> Caractere simples 
%d -> Decimal 
%e -> Notação cientifica
%f -> Ponto flutuante
%o -> Octal
%s -> Cadeia de caracteres
%u -> Decimal sem sinal
%x -> Hexadecimal
%ld -> Decimal longo
%lf -> Ponto flutuante longo
