Uma constante é um dado que é fixo e não pode ser alterado. podemos usar essa pequena instrução como exemplo de uma constante:

printf("Joao tem %d anos", 10);

Além das constantes também existem as variaveis, que são instruções que armazenam um determinado conteudo com base em seu tipo e acessam o mesmo com base no nome que foi especificado.

Existem 5 tipos de variáveis basicas, sendo elas:

char -> 1 byte, escala de -128 a 127
int -> 2 bytes, escala de -32768 a 32767
float -> 4 bytes, escala de 3.4E-38 a 3.4E+38 
double -> 8 bytes, escala de 1.7E-308 A 1.7E+308 

O C também permite que sejam adicionados modificadores, sendo eles:

long ou long int (4 bytes)
unsigned char (de 0 a 255)
unsigned int (de 0 a 65535)
unsigned long 
short (2 bytes)
